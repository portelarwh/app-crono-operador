# Crono Operador

App web simples para cronometragem de operadores.

## Como usar

Abra `index.html` diretamente no browser — sem build, sem dependências.

## Estrutura

```
├── index.html   # estrutura
├── styles.css   # estilo
├── app.js       # lógica
└── assets/      # imagens, favicon, fontes
```
